## Getting Started
Try the same procedure to your our data. Here is how: 

### Prerequisites
- Anaconda
- Rstudio


### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/operator-ita/SmartHome-DataAnalytics
   ```
3. Load our R environment
   ```sh
   conda activate r_env
   ```
